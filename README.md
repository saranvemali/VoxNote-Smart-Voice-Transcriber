# VoxNote — Smart Voice Transcriber 🎤

VoxNote is a modern web-based speech-to-text application that converts spoken words into real-time text using the Web Speech API. It provides a clean animated interface and allows users to easily copy, clear, or export transcripts as PDF or image files.

---

## 🚀 Features

- 🎤 Real-time speech-to-text conversion
- 🔄 Continuous voice recognition
- ✨ Modern animated glassmorphism UI
- 📄 Export transcript as PDF
- 🖼 Download transcript as Image
- 📋 Copy text to clipboard
- 🧹 Clear transcription instantly
- ⚡ Lightweight and browser-based

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Glassmorphism UI + Animations)
- JavaScript
- Web Speech API
- html2canvas
- jsPDF
